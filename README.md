# nhotepad

## About:

Text editor

## Usage:

### Open a file

- nhotepad file

### Options

- `C+q`: quit
- `Arrow up`: move up
- `Arrow down`: move down
- `Arrow left`: move left
- `Arrow right`: move right

### Info about development

#### Stuff that work

- Keyboard handler
- Control + A - Z binds
- Custom binds
- Write characters
- Delete characters
- Open files from command line
- Arrow movement
